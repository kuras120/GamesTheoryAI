# Repository structure

- `src/main/java/com/games/theory/chess` — chess JavaFX controller and model.
- `src/main/java/com/games/theory/tictactoe` — tic-tac-toe UI, game rules, and
  Python integration.
- `src/main/java/com/games/theory/utils` — shared resource and process helpers.
- `src/main/resources` — FXML views and bundled game data.
- `src/test` — unit tests for tic-tac-toe rules and integration parsing.
- `docs/domain` — independently maintained game-domain descriptions.
- `docs/guidelines` — focused development and integration guidance.
- `libs.gradle` — Python package download and virtual-environment build tasks.
